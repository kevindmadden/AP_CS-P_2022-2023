We are going to discuss two main things today:
  • Further discussion of padding, border, and margin
  • CSS Selectors (a more specific way to target elements for styling)

Click here for a very detailed listing of all the different CSS selectors. A CSS selector provides ways to target more specific elements. For example, say you had several paragraphs in paragraph tags on your website. If you wanted additional style elements to make the first and last paragraph stand out, you could use CSS selectors.
